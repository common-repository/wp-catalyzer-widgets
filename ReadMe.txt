== Description ==

WP Catalyzer Widgets is an easy-to-use widgets plugin that helps you add all Social Media Widgets. It adds all Social Media Widgets to any WordPress Plugin.Right now it has Facebook Like Box , Recent Tweets Box, Instagram Feeds, Video Box supporting Youtube, Vimeo and Daliymotion,Google Plus Widget and Flickr Feeds.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'WP Catalyzer Widgets'
3. Activate 'WP Catalyzer Widgets' from your Plugins page.

= From WordPress.org =

1. Download WP Catalyzer Widgets.
2. Upload the 'wp-catalyzer-widgets' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate 'WP Catalyzer Widgets' from your Plugins page.
